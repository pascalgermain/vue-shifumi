# Vue Shifumi

> A Vue.js Shifumi game.

> http://pascalgermain.github.io/vue-shifumi/

## Project setup
```bash
npm install
```

### Compiles and hot-reloads for development
```bash
npm run dev
```

### Compiles and minifies for production
```bash
npm run build
```

### Lints and fixes files
```bash
npm run lint
```
